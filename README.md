# 📁 File Upload Automation

Tự động upload file lên GitHub Releases với GitHub Actions.

## 🚀 Cách sử dụng

### 1️⃣ **Upload tự động khi Push**

1. Tạo thư mục `files/` trong repository:
   ```bash
   mkdir files
   ```

2. Đặt file cần upload vào thư mục `files/`

3. Push lên branch `main`:
   ```bash
   git add files/
   git commit -m "Add files to upload"
   git push origin main
   ```

4. GitHub Action sẽ **tự động tạo Release** với file của bạn ✨

### 2️⃣ **Upload thủ công (Manual Trigger)**

1. Vào tab **Actions** trong repository
2. Chọn workflow **Upload File**
3. Click **Run workflow**
4. Input đường dẫn file (nếu cần)
5. Click **Run workflow**

## 📥 Lấy link download

Sau khi Action chạy xong:

1. Vào tab **Releases** của repository
2. Tìm release mới được tạo
3. Click vào file để download hoặc copy link

**Link tải trực tiếp:**
```
https://github.com/mrtaix/file-upload/releases/download/release-X/filename.ext
```

## ⚙️ Tùy chỉnh

Mở file `.github/workflows/upload-file.yml` để:
- Thay đổi trigger (push, schedule, manual)
- Chỉnh sửa thư mục source
- Tùy chỉnh release notes

## 📌 Tips

- ✅ File được upload lên **GitHub Releases** (công khai)
- ✅ Có link download trực tiếp
- ✅ Tự động backup trên GitHub
- ✅ Không giới hạn dung lượng (tuân theo GitHub Terms)
